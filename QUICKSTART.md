# Быстрый старт Affiliate Bridge

## Установка и запуск

### 1. Установите зависимости

```bash
pip install -r requirements.txt
```

### 2. Инициализируйте базу данных

```bash
python init_db.py
```

Это создаст базу данных и заполнит её тестовыми данными.

### 3. Запустите приложение

```bash
python run.py
```

Приложение будет доступно по адресу: **http://localhost:5000**

## Основной функционал

### Для компаний:
1. Войдите с аккаунтом компании
2. Перейдите в личный кабинет (/dashboard)
3. Создайте новый оффер, нажав "Создать оффер"
4. Просматривайте статистику кликов и продаж

### Для партнёров:
1. Войдите с аккаунтом партнёра
2. Перейдите в каталог офферов (/offers)
3. Выберите интересующий оффер
4. Нажмите "Получить ссылку" для создания партнёрской ссылки
5. Используйте эту ссылку для продвижения
6. Отслеживайте статистику в личном кабинете

## API Endpoints

### Аутентификация
- `POST /api/register` - Регистрация
- `POST /api/login` - Авторизация

### Офферы
- `GET /api/offers` - Список офферов
- `POST /api/offers` - Создание оффера (компании)
- `PUT /api/offers/<id>` - Обновление оффера

### Партнёрские ссылки
- `POST /api/affiliate-links` - Создание ссылки
- `GET /api/affiliate-links` - Список ссылок

### Трекинг
- `GET /track/<code>` - Переход по партнёрской ссылке

### Статистика
- `GET /api/stats/partner` - Статистика партнёра
- `GET /api/stats/company` - Статистика компании
